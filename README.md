# Guestline battleships exercise

![Example of a round](./example.JPG)
![Instructions](./instructions.JPG)

## Setup
```bash
git clone https://github.com/pabrodez/guestline-battleship.git
cd guestline-battleship
npm install
npm run build
```
## Start
```bash
npm run play
```
## Run tests
```bash
npm run test
```
